Small API with my favourite products and their reviews!

Description:

Learn about my tastes, find good products, and read my mostly unbiased reviews! Also includes approximate pricing on each item (prices provided in Canadian dollars), and a final score! Use this API whenever, wherever you go for absolutely no payment!

Functions:

Currently supports three functions - I may add more for Assignment 3

  1. Find all products that I recommend
  2. Read all reviews for all products I have
  3. Find all reviews for a particular product

Usage:

  1. Use the following URL with no input: readAllProducts.php, and get the products I recommend in a JSON encoding back.

  2. Use the following URL with no input: readAllReviews.php, and get the reviews in a JSON encoding back. It automatically clusters the reviews for one product together, so you get all information you need! Note: reviews can have a score of null, this simply means that the most recent score remains.

  3. Use findReviewByProduct.php and provide an ID of a product that you want reviews for. This returns all reviews for a particular product, and allows you to make an informed decision about something! Note: reviews can have a score of null, this simply means that the most recent score remains.
