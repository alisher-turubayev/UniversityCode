Assignment 2

Main Page

The page is scrollable in all cases, accessible from the navigation menu as 'Home', and the content is shrunk to properly fit the screen. In case of desktop, the left navigation menu is on the left and done using the example of Windows desktop application found on the Microsoft website, icons for navigation menu were found on IconShock. In case of iPhone and Google Pixel 2 XL, the navigation bar is located on the bottom, as per the design standards for iOS and Android. Icons were taken from Icons8. The logo was created using Hipster Logo Generator

Store Page

All devices display the scrollable pages (in case of desktop, the page scrolls to the right). The products each are presented on a separate card, to create distinction from the background. Each card expands to display more info about the product. All cards display the name, a picture, and a price of the product. The desktop version also shows promotional banner. All products were found here

Sign-in Page

The desktop version creates a pop-up window to sign up, mobile devices show the sign-up/log-in page before proceding to the Account page. This page shows after a new user presses the 'Account' button on the navigation menu. Password strength recommendations were takes from Microsoft website.

Orders Page

Accessible from the main 'Account' page (not provided in here); hence, it has a back button to return to the main 'Account' page. All orders have their status and the products included in the order displayed.

Messages Page

Done as a help chat. Accessible from the main 'Account' page (not provided in here); hence, it has a back button to return to the main 'Account' page. Messages have different style depending on the OS.

Socials Page

This page is a continuation of the 'Main' page; hence, accessible after scrolling down the main page. The logo and the store name are non-scrollable, and stay in the screen. This page displays icons (in case of phones) to the social media, and latest post on various social media(Facebook, Instagram, Twitter). Default pages are displayed as I don't have accounts for the store.
